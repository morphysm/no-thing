# Panel 1 — Emergence of the Vortex

## Caption

**The wave broke before the sun.**

## Body (expandable)

The unformed cannot disturb itself. The first event is the first asymmetry — the wave that has no source, the rupture that precedes cause. The doctrine names this Sowilo: the sun-rune drawn before the sun exists, the frozen wave-form that announces a sun is coming. Sowilo is not yet light. It is the shape light will take.

## Function

First event. First asymmetry. Forward-flanks Panel 3 (the sun the wave preceded). Introduces the Sowilo-rune as doctrinal anchor without rendering it explicitly.

## Image prompt

1950s religious tract illustration, oil painting, academic realism, Watchtower paradise plate composition, mid-century commercial illustration style, eye-level horizon, naturalistic light, the same empty landscape as before but a wave-form disturbance crosses the sky — angular, jagged, like a frozen ripple, the still water of the lake mirrors it as an asymmetric crease, no figures, no animals, no architecture, the foreground meadow still empty, the first asymmetry in a world that was uniform, painted with the same earnest reverence as a paradise plate, no irony, no drama, only the first occurrence.

## Production notes

- Continuity with Panel 0 is essential. Same horizon line, same hills, same lake, same empty meadow.
- The wave-form disturbance should be subtle — image generators will tend to make it dramatic. Push toward *frozen ripple*, *first asymmetry*, *barely perceptible*.
- The Sowilo-rune shape (jagged zigzag) should be implied in the disturbance pattern but never explicitly drawn as a rune.
